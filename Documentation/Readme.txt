Gyper AR Installation Project
=============================

SETUP
-----

On the PC:

- DISCONNECT THE KINECT

- Install The Kinect SDK v1.5 from: 
http://download.microsoft.com/download/0/B/C/0BC13867-7ECD-4AC8-9400-873FF267467D/KinectSDK-v1.5-Setup.exe

- Follow the process, which should also prompt you to install the Kinect Developer Toolkit from:
http://download.microsoft.com/download/B/B/A/BBA5742E-8E6B-4C05-ADF2-B2F076DD7741/KinectDeveloperToolkit-v1.5.2-Setup.exe

- Connect the Kinect

- Run the application
